ONE-TIME HOST SETUP

This game needs a free Firebase Realtime Database URL so players can join from their own devices. This takes about 2 minutes.

1. Go to console.firebase.google.com
2. Click Add project → give it any name → Continue
3. In the left menu click Realtime Database → Create database
4. Choose any location → Start in test mode → Enable
5. Copy the URL shown (looks like https://your-project-default-rtdb.firebaseio.com)
6. Paste it below
