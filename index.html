ONE-TIME HOST SETUP

This game needs a free Firebase Realtime Database URL so players can join from their own devices. This takes about 2 minutes.

1. Go to console.firebase.google.com
2. Click Add project → give it any name → Continue
3. In the left menu click Realtime Database → Create database
4. Choose any location → Start in test mode → Enable
5. Copy the URL shown (looks like https://your-project-default-rtdb.firebaseio.com)
6. Paste it below

🔧 Fix Fixtures

Updates this game's match list to the official World Cup schedule, keeping the same game code. Everyone's picks are remapped by team, so nobody needs to rejoin or re-pick. A backup is saved automatically first.

⬇️ Import Picks

Copy players' predictions from an older game into this game. Picks are matched by team, so they map correctly even if the fixtures were reordered. You'll see a preview before anything is saved.

👥 Player Team Picks

How many of the 5 per-team picks each player has used. Penalty-shootout wins are free and don't count.

📋 Player Breakdown

Full accounting from stored results. Expected lives = baseline − wrong − (2 × missed) + score bonuses. If Actual differs from Expected, that player had a manual life adjustment (or a drift to investigate).

✏️ Set KO Fixtures

The Round of 32 ties are pre-filled from the current standings. Correct any to match the official bracket (especially the third-place slots), then save. Only available before KO predictions open.

✍️ Enter Scores For Player

For entering exact-score predictions a player gave you (e.g. younger players). Any match that hasn't kicked off yet is shown — even in groups already underway. The same limits apply: max 12 scores, one per group.
